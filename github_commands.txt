$ git status
# this command will tell you the untracked or tracked files in your folders, red is untracked, while green is tracked and ready to be commited/pushed
$ git add .
# this add all the new files to the commit
$ git commit -am 'write some message'
# this command adds an explaination on what changes you've made to the files
$ git push
# updates the stages changes to your github account, if it asks for the password for an ssh key passphrase, just press enter, there is no password
